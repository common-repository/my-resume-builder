My Resume Builder allows you to create a beautifully formatted Resumes in minutes.

== Description ==

My Resume Builder is a wordpress plugin that helps you create and easily edit Resumes in minutes.

In the Resume editor page, you have access to functionalities that allows you to enter Personal, Education History and Professional
information. 

A WYSIWYG (What You See Is What You Get) text editor is included for you to enable you set how your Resume will be displayed
after you have finished editing it in the Resume view page.

= Main Features =
* Adding a contact information
* Attach a photo to your Resume
* Drag and Drop blocks to reorder them
* Add Education and Work History
* Share your Skills
* Share with others Project that you have worked on
